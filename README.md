# GenAdapt-FM @ NeurIPS 2026

Website for the NeurIPS 2026 workshop **Generalization and Adaptation in the Era of Foundation Models**.

Static site — open `index.html` in a browser, or serve the folder (e.g. `python3 -m http.server`).

Pages: `index.html` (home), `call-for-papers.html`, `program.html`, `organizers.html`.

Content is a tentative draft based on the workshop proposal.
